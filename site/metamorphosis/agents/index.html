    Agents Package¶

    The metamorphosis.agents package includes LangGraph-based agent workflows for multi‑stage text processing. It offers simple synchronous and streaming examples via a small FastAPI service.

    Core Components¶

    1. FastAPI Service (agent_service.py)¶

    The main service provides REST API endpoints for text processing workflows.

    2. LangGraph Workflow (self_reviewer/)¶

    Implements a clear multi-step learning example.

    %%{init: {'theme': 'base', 'themeVariables': { 'primaryColor': '#e0f2ff', 'edgeLabelBackground':'#ffffffaa', 'clusterBkg':'#f7fbff', 'fontFamily':'Inter'}}}%% graph LR A[START] --> B[Copy Editor] B --> C[Summarizer] B --> D[Word Cloud] B --> E[Achievements Extractor] E -->|if tools| F[ToolNode: extract_achievements] F --> G[After Achievements Parser] G --> H[Review Text Evaluator] H -->|if tools| I[ToolNode: evaluate_review_text] I --> J[After Evaluation Parser] C --> K[END] D --> K J --> K
